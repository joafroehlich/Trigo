Berechnen Sie den Winkel α, wenn a = 14,3 m, 
b = 26 m und γ = 82,1°.

Seite b) vor. Der Fall hat dann eine eindeutige 
Lösung, wenn γ < 180° ist.

c2 = a2 + b2 - 2 * a * b * cos γ

c2 = 14,32 + 262 + 2 * 14,3 * 26 * cos 82,1°

c2 = 14,32 + 262 - 2 * 14,3 * 26 * 0,1374
